5 steps to unlimited AI

01

Clone the Draftly Studio repo

Download the open-source studio — front-end, node canvas, and local AI server included.

git clone https://github.com/piyushxt43/draftly-studio-.git
02

Install Python + CUDA

Install PyTorch with GPU support. Requires Python 3.10+ and an NVIDIA GPU with CUDA.

pip install torch torchvision --index-url https://download.pytorch.org/whl/cu121
03

Download your AI models

Install Hugging Face libraries. Models auto-download on first use (~5-15 GB each).

pip install diffusers transformers accelerate safetensors
04

Start the local server

Runs at localhost:8000. The studio auto-detects the local server and shows "Local" models.

cd local-server && python server.py
05

Open the Studio and generate

The studio opens with a ready-to-use workflow. Select local models, hit Generate — zero cost, unlimited.

npm install && npm run dev → Open localhost:3000
